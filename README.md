# bit_log 

**Визуализатор работы логических элементов на Си**

Проект создан для наглядного изучения работы логических элементов (AND, OR, XOR сдвиги).
Я новичок пришла идея будущего проекта. 

### Как это работает:
Программа принимает два числа и логический элемент.
1. Вычисляет результат.
2. Раскладывает каждое число на биты.
3. Визуализируем процесс в концоли.

### Поддерживаемые операции:
* '&' - И
* '|' - ИЛИ
* '^' - Исключающее ИЛИ

### План развития 
- [] Добавить сдвиги ( <<  >> )  
- [] Добавить инвертор ( ~ )
- [] Цветную индикацию. 
- [] Бесконечный цикл многое другое.
